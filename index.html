自傳(Autobiography)

生性樂觀、開朗型內向人，喜歡從做中學，靈活運用知識，不墨守成規死讀書，只要是對有興趣的，觀察、吸收力就非常強。

【第一份工作－網站工程師】

進公司後，感謝主管讓我有三個月扎實的新人訓練，學了一些在學校比較沒機會碰的技術，像是 Docker、Symfony，也更完整的學了一次 Git 版控，比較正式的了解 PSR-2，奠定了之後開發上的流暢度。

開發時，有學到 FogBugz、Gerrit、Gitlab、Jenkins、Redis、Node.js…，差不多是這時候開始，比較常會使用到 VS Code。

後來公司新增服務項目，組織進行調整，改負責區塊鏈相關技術，因此學了一些 Web3 的知識，不過這個領域真的要花費較多心力持續接收新資訊，上鏈的資料要盡可能的嚴謹，因為可以說是對所有人開放的，舉例來說，如果區塊鏈有升級，我們必須要跟上，不然可能會導致我們的帳本只能同步到分叉的高度。

其中還有大約半年的時間，我們被加派支援一個專案，算是比較完整的從底層架構設計資料庫到 API，讓我比較有參與大型專案的研發的感覺，過程中需要跟企劃、前端工程師、資料庫工程師跨部門，甚至是同部門同仁間持續進行溝通，期間積極導入較新、嚴謹的語法，並提議優化共同開發流程。

【家庭概況】

父親做過研發工程師、品管、研發課長、導遊，如今已退休，每次父親要求我們要做到什麼時，都會先以身作則；而母親從事業務，自從生下二姊以後，便開始從事家庭主婦，打點家中大小事；兩個姊姊目前都在臺中工作。

從小父母親就要求我們要腳踏實地、做事追根究底，凡事要有實事求是的精神，讓我們自己選一樣家事來做，每個禮拜輪流煮一餐，藉此更懂得付出才有收穫的道理；上大學後，便讓我們學習為自己的花費負責，因此我養成記帳的好習慣。

【班級幹部】

在高職時，擔任了三任副班長，積極處理班上事務，可以帶領著班級前進，讓我感覺到了強烈的使命感；也當了兩任的器材管理員。大學則是當了兩任服務股長，一任的衛生股長。

【成長及學習過程】

小時候，因為家中沒有電視，所以平常休閒時會看書，可以一整天都在讀同一本書，那時家中有台 Windows 2000 的電腦，經常會壞掉，就常常自己嘗試著修理、重灌，真的解決不了就會上網去 Yahoo 奇摩知識+ 發問、跑到書局、圖書館查閱書籍或相關資料，偶而會試著回答問題，後來在高職擔任過圖書館志工，因此對電腦相關的事就非常有興趣。

高職時，實習成績還算不錯，因此代表科上參加工科技藝競賽，雖說沒達到期望的比賽成績，有點對不起那些實習成績也不錯的同學，不過這激勵了我努力拚統測。

大學時期，會不定期請業界人士、別校老師、業師、學長姐來演講、上課，在實驗室生活，有時候會遇到畢業的學長姐回來，常常會分享一些有趣的內容。

當完兵後，很幸運在當天就收到面試邀約，不過因為當兵前去上的駕訓班還沒上完，所以只能多等一個月，考完駕照才進公司。

【未來期許】

縱觀從小到大，父母、班導師、主管幾乎都是好 Mentor，如今得利於網際網路的發展，還可以藉此接觸到許多不認識的 Giver，都扶持著我走上『相對』正確的道路，也感謝一路上的益友、損友，無不讓我學習到許多。

從 2020 年開始，因為疫情導致線下活動急遽減少，如今貌似有復甦的景象，我會繼續多參與活動、講座，期許之後的路，可以持續累積實力，並擁有足夠的運氣，可以不忘初衷的達成目標，從自身出發，冀望能帶給社會些微的貢獻。

【實作】

國中升高職時我基於興趣填了資訊科，那時我開始接觸到網頁以外的程式語言，那時候 Facebook 上流行著 Battle(俄羅斯方塊)，我就在高一的時候寫了一個俄羅斯方塊，並參考網路上的方法把它改成了連線板，我也就開始對程式產生了興趣，後來陸陸續續有寫一些程式，像是踩地雷、聊天室、五子棋、FTP客戶端。

大二時，參加系上網頁程式設計比賽，很幸運的獲得第一名，其中花最多時間作的是其中的檔案管理系統 [ file-manager ] ，利用大一所學的 PHP 搭配之前寫 FTP 客戶端的經驗，組合成一個具基本檔案管理的網頁。

因為在系辦工讀的原因，隔年被問要不要改寫系網，於是乎我就當了幾個月的系網網管，我最後是以 WordPress 為底 [ KUAS_CSIE_Wordpress_Themes_TCC-Department ] ，支援語言切換，而在那之前有先用 CodeIgniter 寫過一個先行版 [ kuas-csie ] ，但基於安全考量，還是決定用 WordPress。

大三上學期後半，和高第一同學合作，做一個用 Arduino 控制的電源插座。

寒假結束後我參加資研社兩天的 Hackathon ，用 Python(後台) + Html5 刻一個線上電子白板 [ White-Brain ] ，基本畫線、拉直線、直線預覽、上／下一步等等。

大三升大四在學校實習，做了一個網頁版會員訓練系統 [ TcTube ] ，主要都在研究 HTML5 影片播放。

大四上修 Pygame 期末作業，做了一個倉庫番遊戲 [ PygameSokoban ] ，除了一般正常的推箱子，還可以用滑鼠控制。

大四下則是在 Android 課程中做了終極井字棋 [ UltimateTicTacToe ] ，而我們採用的勝利條件是用佔領下來的地盤在大的九宮格連成一條線，一局大約 15 分鐘到半小時。

而快畢業時，我有接一家同學介紹的花店網站 Case，是一般的購物車網站，但我不大會用最後付款取貨的部分。

畢業後，在等入伍前我也重構了大一寫的檔案管理系統 [ TcStorage ] ，而當兵時也有被拉去出資訊公差，幫忙灌電腦、拉電話網路線、設定網路等等，多學了一兩招網管技能。

在職期間，組隊以【不能逃 不能逃】參加公司為期三天的區塊鏈 Hackathon 比賽，獲得最佳潛力獎，作品是票券系統，可以應用在演唱會門票上，主要發想是想要解決一般人搶不到票，黃牛滿天飛的情形。主要負責智能合約的部分，對應到一般網站系統，大致上可以想作是後端的部分。
